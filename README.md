<h2 id="3">앱 기능 설명</h2>

-   녹음 기반의 밴드 음악 연주 및 커버 플랫폼
-   시간적, 실력적 부담 없이 밴드 음악을 즐길 수 있음
-   다양한 이펙터를 제공함으로 재미있게 커버할 수 있음
-   장비가 없더라도 녹음한 커버에 대하여 각종 정제가 들어가서 장비 없이도 고퀄리티의 녹음이 가능
-   다양한 세션들을 조합하여 리스너의 취향대로 듣기 가능
